# Changelog

All notable changes to SpeakFlow will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [v1.0.0] - 2026-05-08

### Added
- 🎙️ AI-powered English conversation with Google Gemini 1.5 Flash
- 📝 Real-time grammar correction with highlighted mistakes
- 📊 Progress dashboard with weekly speaking score chart
- 📚 8 practice topic categories (IELTS, Business, Travel, etc.)
- 🔥 Daily streak tracker and activity calendar
- 🏆 Achievement badge system
- 🌙 Full dark mode support
- 🚀 Animated onboarding (3 slides)
- 🔄 Android In-App Update system (Flexible + Immediate)
- 🔥 Firebase Remote Config for version control
- 🛠️ Maintenance mode screen
- 🎨 Premium Material 3 UI with purple/blue gradient theme
- 🔔 Beautiful update dialogs with release notes
- 👤 Profile screen with level progress and subscription status

### Technical
- Flutter 3.x with Material 3
- Riverpod state management
- GoRouter navigation with shell routes
- fl_chart for interactive bar charts
- flutter_animate for smooth animations
- google_generative_ai SDK integration
- in_app_update for Play Store updates
- Firebase Remote Config for backend control
